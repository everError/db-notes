### **MySQL**

---

#### **개요**

MySQL은 Oracle에서 개발 및 관리하는 오픈소스 관계형 데이터베이스 관리 시스템(RDBMS)입니다. 웹 애플리케이션 개발에 널리 사용되며, LAMP 스택(Linux, Apache, MySQL, PHP)의 핵심 구성 요소로 유명합니다. 오픈소스 기반으로 커뮤니티 에디션을 무료로 사용할 수 있어 스타트업부터 대규모 엔터프라이즈까지 폭넓게 채택되고 있습니다.

#### **주요 특징**

- **관계형 데이터 모델**: 테이블, 열, 행으로 구성된 정형 데이터를 관리합니다.
- **스토리지 엔진 선택**: InnoDB, MyISAM 등 목적에 맞는 스토리지 엔진을 테이블 단위로 선택할 수 있습니다. 기본 엔진인 InnoDB는 트랜잭션과 외래 키를 지원합니다.
- **높은 성능**: 읽기 중심의 워크로드에서 뛰어난 성능을 발휘하며, 대규모 웹 서비스 환경에 적합합니다.
- **복제 및 고가용성**: Master-Replica 복제, Group Replication, InnoDB Cluster 등을 통해 고가용성 구성을 지원합니다.
- **광범위한 플랫폼 지원**: Windows, Linux, macOS 및 Docker 컨테이너 환경을 지원합니다.
- **오픈소스 생태계**: 활발한 커뮤니티와 풍부한 서드파티 도구 생태계를 보유하고 있습니다.

#### **주요 데이터 타입**

- `VARCHAR`, `TEXT`: 가변 길이 문자열. `TEXT`는 대용량 문자 데이터를 저장합니다.
- `INT`, `BIGINT`: 정수형 데이터.
- `DECIMAL`, `FLOAT`: 소수점 데이터. `DECIMAL`은 정밀한 수치 계산에, `FLOAT`는 근사값 처리에 사용됩니다.
- `DATETIME`, `TIMESTAMP`: 날짜 및 시간 데이터를 저장합니다. `TIMESTAMP`는 시간대 변환을 자동으로 처리합니다.
- `JSON`: JSON 형식의 데이터를 저장하고 쿼리할 수 있습니다. (MySQL 5.7.8 이상)

#### **주요 관리 도구**

- **MySQL Workbench**: MySQL 공식 GUI 도구로, 데이터베이스 설계, 쿼리 실행, 서버 관리 기능을 제공합니다.
- **MySQL Shell**: SQL 및 JavaScript, Python 스크립트를 지원하는 고급 CLI 클라이언트로, InnoDB Cluster 관리에 활용됩니다.
